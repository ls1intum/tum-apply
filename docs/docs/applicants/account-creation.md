# Account Creation

You can browse all open job postings on TUMApply without an account. An account is only required if you want to apply
for
positions or manage your personal data.

Applicants can create an account using a simple and secure email-based registration
process. ([Video](https://live.rbg.tum.de/w/artemisintro/67995))

## How to Register

1. Click the **Login** button in the upper right corner.
2. In the dialog that opens, click on **Register here** at the bottom.
3. Enter your **email address**.
4. You will receive a **verification code by email**.
5. **Enter the confirmation code** to verify your email address.
6. After verification, you can optionally enter a **First name** and a **Last name**.
7. You may also **optionally set a password**.

## Important Notes

- If you **do not set a password** during registration, you can **only log in using the "Email and Code" method**.
- If you **set a password**, you can later log in using:
  - **Email and Password**
  - **Email and Code**
- The verification code is always required for the **initial registration**.
- You can add or change your personal information later in your profile settings.

## Registration During Application

If you start an application without having registered yet, an account will automatically be created during the process:

1. Enter your email address.
2. Receive and confirm a verification code by email.
3. Provide personal details (e.g. first name, last name).
4. After successful verification, you are logged in automatically and can continue uploading documents and writing
   your application texts.
5. Any additional data provided by external login providers will be imported automatically.
