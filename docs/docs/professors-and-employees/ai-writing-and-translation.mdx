# AI Writing & Translation

Every doctoral position on DocApply is published in **both English and German**, allowing
applicants to read the description in their preferred language. To support this, the
**Position Details** step of the [Job Creation Form](./create-job) provides an AI assistant that
**writes, improves, and translates** the job description.

The description is written once in the language of your choice, and the assistant produces the
second language version, removing the need to translate the text manually.

:::info[Both languages are required]
A position can only be **published** once both the English **and** the German description have been provided.
:::

## ✨ What the AI Assistant Can Do

### Write or rewrite the description

Use the AI assistant to draft a description from the details you have already entered (such as
title, research area, and location) or to polish text you have started yourself:

- **Write with DocApply** generates a first draft of the description in the currently selected
  language.
- **Rewrite with DocApply** improves and restructures the text you already have.

The text streams into the editor as it is generated. When it is finished, you can edit it freely.

:::tip
The AI works best when you have already filled in the basic position details (title, research
area, field of studies, location) — it uses them as context to produce a more relevant draft.
:::

### Automatic translation between English and German

You only write the description **once**. Whenever you finish writing or editing one language
version, the assistant **automatically translates it into the other language** in the background.

- The translation appears in the other language tab — switch the language toggle to review it.
- If you keep editing the source text, the assistant restarts the translation with your latest
  version, so the two language versions stay in sync.

:::note
The translation is a starting point, not a final proofread. Always **switch to the other language
and review the result** before publishing — check terminology, tone, and any field-specific
wording.
:::

---

## 🔀 Turning the AI Assistant On or Off

The AI assistant is **optional**. You can enable or disable it from the toggle in the editor
header, and the info icon next to it explains what each mode does:

- **AI enabled**: The assistant helps you draft the description, and the translation into the
  second language is generated automatically.
- **AI disabled**: The assistant is not available, and you must **write and translate both
  language versions manually**.

## ⚙️ Step-by-Step: Creating a Bilingual Description

**Goal:** Produce a complete English and German job description ready for publication.

1. Open the **Job Creation Form** and go to the **Position Details** step (see
   [Create Job](./create-job)).
2. Choose the language you want to write in using the **English / German** toggle.
3. Write the description yourself, or click **Write with DocApply** to generate a draft and
   **Rewrite with DocApply** to improve it.
4. Wait for the **Translating** indicator to finish — the assistant translates your text into the
   other language automatically.
5. **Switch the language toggle** to the other language and **review the translation** for
   accuracy, terminology, and tone. Edit it directly if needed.
6. Make sure **both** language versions are complete and within the character limit.
7. Continue through the remaining steps and **publish** the position.
