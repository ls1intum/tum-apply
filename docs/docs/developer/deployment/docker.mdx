# 🐳 Docker

## Docker Compose Support

To launch required third-party services, JHipster generates Docker Compose files in `src/main/docker/`.

Start services:

```bash
docker compose -f docker/local-setup/services.yml up -d
```

Stop and remove containers:

```bash
docker compose -f docker/local-setup/services.yml down
```
